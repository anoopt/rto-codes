# Andaman and Nicobar Islands RTO Data

This folder is set up for Andaman and Nicobar Islands RTO codes but doesn't contain any RTO data yet.

## How to Contribute

To add RTO data for Andaman and Nicobar Islands, create JSON files following this pattern:

### File Naming
- Format: `an-NN.json` (e.g., `an-01.json`, `an-02.json`)
- Use lowercase for the state code
- Use 2-3 digit numbers

### Required Fields
- `code` - RTO code (e.g., "AN-01")
- `region` - Area/region name
- `city` - City name
- `state` - "Andaman and Nicobar Islands"
- `stateCode` - "AN"
- `district` - District name
