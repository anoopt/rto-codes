# Dadra and Nagar Haveli and Daman and Diu RTO Data

This folder is set up for Dadra and Nagar Haveli and Daman and Diu RTO codes but doesn't contain any RTO data yet.

## How to Contribute

To add RTO data for Dadra and Nagar Haveli and Daman and Diu, create JSON files following this pattern:

### File Naming
- Format: `dd-NN.json` (e.g., `dd-01.json`, `dd-02.json`)
- Use lowercase for the state code
- Use 2-3 digit numbers

### Required Fields
- `code` - RTO code (e.g., "DD-01")
- `region` - Area/region name
- `city` - City name
- `state` - "Dadra and Nagar Haveli and Daman and Diu"
- `stateCode` - "DD"
- `district` - District name
